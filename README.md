I am a Software Engineering undergraduate at **Pune Institute of Computer Technology**, building at the intersection of machine learning research and production-grade software systems. My engineering philosophy is rooted in measurable outcomes — I do not ship code, I ship impact metrics.

My work spans the full lifecycle of intelligent systems: from raw data ingestion and feature engineering to model training, evaluation, and deployment into concurrent backend architectures. I have applied **CNN and Vision Transformer (ViT)** architectures to real-world remote sensing data, engineered fraud detection pipelines in fintech hackathons, and architected full-stack platforms that serve live user bases. Every project I ship is driven by a well-defined problem statement, a rigorous experimental baseline, and a quantified result.

On the software engineering side, I design modular **REST APIs**, secure JWT-authenticated backends, and scalable concurrent service architectures. I write clean, maintainable code in **C++, Java, and Python**, with a growing depth in ML frameworks and LLM application stacks including **LangChain, RAG pipelines, and multi-modal inference**.

I bring a **product engineering mindset** to every codebase — thinking about users, performance at scale, and the system boundaries that make software trustworthy over time.
